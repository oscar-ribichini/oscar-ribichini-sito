# Archivio rassegne notizie AI

Contiene tutte le rassegne giornaliere generate dalla skill `/new-ia`.

## Convenzione nomi file

```
YYYY-MM-DD.md
```

Esempio: `2026-05-05.md`

## Come archiviare

Dopo aver eseguito `/new-ia`, chiedi a Claude: "salva la rassegna di oggi" oppure "archivia le notizie". Il file viene creato in questa cartella con la data odierna.

## Struttura di ogni file

- Data di riferimento
- Notizie divise per categoria (Italia & Europa, Mondo, Tecnologia, Ricerca, Normativa)
- Per ogni notizia: titolo, fonte, descrizione, link
