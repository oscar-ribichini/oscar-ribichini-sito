# 1. RIASSUNTO (per descrizione video)

In questa lezione introduttiva scopriamo le differenze fondamentali tra Claude Code e i chatbot tradizionali come ChatGPT, Gemini e Grok. Claude Code non è una chat — è un vero ambiente di lavoro installato sul computer, che lavora direttamente sui tuoi file in locale, mantiene un contesto di progetto permanente e si personalizza in modo profondo. Le skill di Claude Code sostituiscono i GPT e le gemme con mattoncini riutilizzabili e potenti. Una lezione teorica essenziale prima di sporcarsi le mani con la pratica.


# 2. TESTO PULITO

Voglio mostrarvi subito le differenze principali tra Claude Code come ambiente e l'utilizzo tradizionale dei chatbot di intelligenza artificiale generativa. Quando dico ChatGPT, intendo tutti gli strumenti che usavate prima: Gemini, Grok, Claude in modalità chat, e simili.

Come lavora con i file

I chatbot tradizionali vivono nel browser. Claude Code invece è installato fisicamente sul computer — questa è la prima differenza importante.

Di conseguenza, i chatbot rispondono nella chat e tutto quello che produci deve essere copiato e incollato fuori. Claude Code, lavorando in locale, salva tutto direttamente sul disco: ogni file che crea è già sul vostro hard disk, senza download manuali.

I chatbot tradizionali sono sequenziali: una cosa alla volta, un file alla volta, una chat alla volta. Claude Code permette più sessioni e più file in contemporanea.

Il contesto di progetto

Questa è la differenza più importante. I chatbot tradizionali partono da zero a ogni nuova chat. Anche con la memoria abilitata, il grosso delle informazioni viene perso.

Claude Code ha il contesto del workspace. Significa che ha accesso permanente alle vostre cartelle di lavoro, ai vostri file, ai vostri progetti. Se gestite dieci clienti e state lavorando sul cliente numero quattro, Claude Code carica solo il contesto di quel cliente — non brucia memoria inutilmente sugli altri.

Personalizzazione

I chatbot tradizionali si personalizzano con le custom instructions: poche righe, tono di voce, qualche preferenza. Claude Code permette una personalizzazione senza limiti, dinamica, diversa per ogni cliente o progetto. Il vostro workspace sarà completamente diverso da quello di chiunque altro.

Con ChatGPT si usavano prompt lunghissimi, anche di due pagine, perché bisognava ridefinire ogni volta il contesto. Con Claude Code questa cosa scompare quasi del tutto: il contesto è già lì.

Le skill

Le skill sono mattoncini riutilizzabili del vostro lavoro. Se fate spesso preventivi, create una skill per i preventivi. Se sbobbinate le call con i clienti, create una skill "Sbobbinami". Se ogni settimana cercate le news di settore, create una skill "News di settore".

Sono molto più potenti dei GPT e delle gemme, estendibili e personalizzabili. Claude Code è una cassetta degli attrezzi, non una chat.

Conclusione

ChatGPT è una chat. Claude Code è un ambiente di lavoro. Non state sostituendo una chat con un'altra chat — state passando a un modo di lavorare completamente diverso.
